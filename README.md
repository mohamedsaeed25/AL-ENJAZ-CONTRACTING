مشروع نظام إدارة شركات المقاولات
=================================

هذا المشروع عبارة عن مثال كامل (فرونت إند + باك إند) لنظام بسيط لإدارة شركات المقاولات.

### المكونات

- backend: خادم Node.js مع Express و TypeScript يقدم REST API بسيطة لإدارة المشاريع والعملاء.
- frontend: تطبيق React + TypeScript (Vite) لواجهة لوحة التحكم.

### خطوات التشغيل (اختصارًا)

1. ادخل إلى مجلد الباك إند:
   - `cd backend`
   - `npm install`
   - `npm run dev`

2. ادخل إلى مجلد الفرونت إند في نافذة أخرى:
   - `cd frontend`
   - `npm install`
   - `npm run dev`

ثم افتح المتصفح على العنوان الذي يظهر لك في الطرفية لتجربة النظام.
